  • Wepify

    About the project

    Wepify ist ein moderner Website-Builder, mit dem man einfach und schnell eigene Webseiten erstellen kann – ganz ohne Programmierkenntnisse.

    Wepify bietet eine benutzerfreundliche Oberfläche, mit der Inhalte individuell gestaltet und flexibel angepasst werden können.

    How I have organized my work process

    Für das Projekt habe ich zuerst die passenden Werkzeuge und Technologien ausgewählt. Dann habe ich geplant, welche Komponenten benötigt werden und welche davon wiederverwendbar sind.

    Dabei habe ich eine weit verbreitete, feature-basierte Architektur verwendet, um den Code übersichtlich und wartbar zu gestalten. Außerdem habe ich bewährte Best Practices angewendet, damit der Code sauber, gut strukturiert und leicht erweiterbar bleibt.

  • Lovit

    About the project

    Lovit.js ist eine JavaScript-Bibliothek, mit der man Fehler in Anwendungen klar und geordnet behandeln kann. Sie sorgt dafür, dass man den Überblick behält und Fehler nicht chaotisch im ganzen Code verteilt sind.

    Lovit macht es leichter, den Code sauber zu halten, und passt sich gut an verschiedene Projekte an – egal ob im Browser oder auf dem Server. Dadurch wird die Entwicklung angenehmer und zuverlässiger.

    How I have organized my work process

    Zuerst habe ich ermittelt, wie die Fehlerbehandlung in JavaScript einfacher und flexibler gestaltet werden kann, damit sie in verschiedenen Situationen gut funktioniert.

    Danach habe ich die wichtigsten Funktionen definiert, die die Bibliothek in der ersten Version enthalten soll.

    Anschließend wurden alle Aufgaben in kleinere Schritte aufgeteilt und sowohl die Bibliothek als auch die Dokumentationsseite sorgfältig geplant.

    Da es sich um eine Bibliothek handelt, war es besonders wichtig, sicherzustellen, dass sie in unterschiedlichen Umgebungen zuverlässig läuft.
